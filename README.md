# Emparejados

Emparejados es una aplicación de citas para Android, desarrollada como parte del curso Proyecto Avanzado Android. El objetivo es permitir a los usuarios conocer nuevas personas mediante un sistema de deslizamiento de perfiles.

## Estado actual

Esta es una versión inicial que contiene:
- Interfaz básica con imagen de perfil, nombre y botones de acción.
- Lógica de navegación simple para pasar perfiles.
- Diseño adaptable a nuevas funcionalidades futuras.

## Tecnologías utilizadas

- Java
- Android XML layouts
- Android SDK

## Próximas funcionalidades

- Registro de usuario
- Sistema de match
- Chat entre emparejados
- Integración con Firebase
