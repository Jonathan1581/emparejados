# Emparejados

## Descripción del proyecto

Emparejados es una aplicación móvil desarrollada para la plataforma Android, cuyo objetivo es ayudar a las personas a encontrar pareja de manera práctica, intuitiva y segura. La app estará basada en una mecánica de deslizamiento similar a la de Tinder, permitiendo a los usuarios mostrar interés en otras personas y hacer "match" cuando el interés es mutuo.

---

## Exposición del problema

Muchas personas desean encontrar pareja, pero se enfrentan a barreras como falta de tiempo, timidez o escasas oportunidades para conocer gente nueva. Si bien existen plataformas de citas, muchas son poco amigables, inseguras o no están adaptadas a ciertos públicos. Emparejados busca ofrecer una solución moderna, accesible y confiable para quienes desean conocer personas con intereses compatibles.

---

## Plataforma

La aplicación será desarrollada inicialmente para el sistema operativo Android, utilizando tecnologías como Kotlin o Java, y Firebase como backend para autenticación, base de datos en tiempo real y almacenamiento de imágenes.

---

## Interfaz de usuario e interfaz de administrador

### Interfaz de usuario
- Registro con correo, número de celular o Google.
- Creación de perfil con fotos, biografía y datos de interés.
- Visualización de perfiles por cercanía o preferencias.
- Deslizamiento para dar "Me gusta" o "Pasar".
- Chats entre usuarios que hayan hecho "match".

### Interfaz de administrador
- Panel de control para moderar perfiles sospechosos.
- Administración de reportes.
- Estadísticas básicas: usuarios activos, cantidad de matches, etc.

---

## Funcionalidad

- Registro e inicio de sesión.
- Creación y edición de perfil.
- Filtros por edad, ubicación e intereses.
- Deslizamiento tipo Tinder.
- Notificaciones en tiempo real al hacer match.
- Sistema de mensajería entre emparejados.
- Reporte de usuarios por mal comportamiento.
